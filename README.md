# IT Lake
This is the backend for the Great IT Lake service

### QUICK START
Run
```shell script
cd docker
docker-compose up
cd ..
gradle bootRun
```
Explore
```http request
GET localhost:8080/
```

### DEPLOYED INSTANCES
Dev stand
```http request
https://it-lake.herokuapp.com/
```
